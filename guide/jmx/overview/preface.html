Preface

The JavaTM Management Extensions (JMXTM) Technology Overview provides an introduction to JMX technology. JMX technology is a new feature in version 5.0 of the Java 2 Platform, Standard Edition (J2SETM platform).

Before You Read This Book

To make full use of the information in the Java Management Extensions (JMX) Technology Overview, you should have knowledge of the following technology:

  • The Java programming language and its development environment
  • The Java 2 Platform, Standard Edition (J2SE) software

Some sections of this overview refer to the following protocols and specifications. You might want to investigate these areas further to gain a deeper insight into JMX technology:

  • The JavaBeansTM model
  • Remote Method Invocation (RMI)
  • Lightweight Directory Access Protocol (LDAP)
  • Service Location Protocol (SLP)
  • JiniTM Network Technology
  • Java Naming and Directory InterfaceTM (JNDI) API
  • Java Secure Socket Extension (JSSE)
  • Java Authentication and Authorization Service (JAAS)
  • Simple Authentication and Security Layer (SASL)

How This Book Is Organized

This document describes the JMX technology in the following chapters:

  • Chapter 1, "Introduction to JMX Technology" provides a brief outline of the JMX technology and indicates its main advantages and features.
  • Chapter 2, "JMX Architecture"
  • Chapter 3, "Instrumenting Your Resources for JMX Technology"
  • Chapter 4, "Using JMX Agents"
  • Chapter 5, "Using JMX Connectors to Manage Resources Remotely" describes the required connector defined by the JMX technology specification. It also outlines an optional generic connector. This chapter includes information on connector security features.
  • Chapter 6, "Discovery and Lookup Services" describes briefly how to use existing discovery and lookup services with the JMX technology
  • Appendix A, "JMX Technology Versions" defines the versions of the Java Specification Requests (JSRs) relating to JMX technology implemented in this version of the J2SE platform.
